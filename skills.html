Дизайн
Figma
Это кросс-платформенный онлайн-сервис для дизайнеров интерфейсов и веб-разработчиков. Figma позволяет разрабатывать интерфейсы в онлайн-приложении. У Figma две ключевые особенности: доступ к макету прямо из окна браузера и возможность совместной работы над документами.
Adobe Photoshop
Это многофункциональный графический редактор. В основном работает с растровыми изображениями, однако имеет некоторые векторные инструменты. Отлично подходит для работы с макетами и разработки веб-приложений.

Programming
JavaScript
Это язык программирования, создан, чтобы «сделать веб-страницы живыми». Например, в браузере JavaScript может: добавлять новый HTML-код на страницу, изменять существующее содержимое, модифицировать стили, также может реагировать на действия пользователя, щелчки мыши, перемещения указателя, нажатия клавиш и многое другое.
Python
Это язык программирования, который входит в число самых популярных на рынке IT. Его используют для создания разных продуктов, от коммерческих до научных. На нем пишут плагины и сценарии. С его помощью сисадмины автоматизируют выполнение серверных задач. Язык помогает создавать приложения, сайты, ботов, онлайн-игры и многое другое.
Git
Это распределённая система управления версиями, которая даёт возможность разработчикам отслеживать изменения в файлах и работать совместно с другими разработчиками.
Gulp
Это инструмент автоматизации front-end разработки. Он помогает автоматизировать рутинные задачи и ускоряет работу над проектом. Например: минимизирует HTML и CSS файлы, оптимизирует изображения, автоматически удаляет неиспользуемый код, а также проверяет его правильнось.
SQL
Это язык структурированных запросов, используемый для работы с базами данных. С помощью SQL можно задавать структуру базы, организовывать выборку данных из неё по каким-то определённым критериям, удалять данные.
PHP
Это язык программирования,который ориентирован на разработку веб-приложений и веб-сайтов. Значительным отличием PHP от какого-либо кода, выполняющегося на стороне клиента, например, JavaScript, является то, что PHP-скрипты выполняются на стороне сервера. PHP позволяет создавать качественные Web-приложения за очень короткие сроки, получая продукты, легко модифицируемые и поддерживаемые в будущем.

Работа с CMS
Wordpress
Это одна из наиболее популярных CMS в мире. С ее помощью можно создавать сайты различного характера и управлять ими без знаний и навыков программирования. Ее также называют движком ресурса. Сегодня в сети существует множество различных программ или систем управления, но WordPress является самой популярной. Более 30 % всех сайтов в интернете работают на данной платформе. А если говорить исключительно о площадках, работающих на CMS, то на WP приходится более 60 %, что делает ее однозначным лидером рынка.
OpenCart
Это быстрая и не требовательная к ресурсам сервера CMS. Это — движок интернет-магазина, ориентированный на создание удобных и продающих интернет-магазинов. Это — очень надёжное и простое решение для желающих создать свой онлайн магазин при минимальных затратах.

Верстка
HTML5
Это язык разметки гипертекста в сети интернет. Простым языком, если CSS определяет внешний вид web-страницы, HTML формирует ее структуру (скелет) посредством заголовков, списков и других подобных элементов, начиная от начала страницы — хедера, и до ее конца — футера.
CSS
Это формальный язык, служащий для описания оформления внешнего вида документа, созданного с использованием языка разметки HTML.
SCSS
Это "диалект" языка SASS, предназначенный для упрощения создания CSS-кода.

Фреймворки
Django
Это чрезвычайно популярный и полнофункциональный серверный веб-фреймворк, предназначенный для создания веб-приложений на языке Python. Веб-фреймворк Django используется в таких крупных и известных сайтах, как Instagram, Disqus, Mozilla, The Washington Times, Pinterest, YouTube, Google и др.

Веб-аналитика
Google Analytics
Это разработанный поисковой системой Google специальный аналитический инструмент для отслеживания поведения пользователей на сайте.
Google Adsense
Это cервис контекстной рекламы и партнерская программа от поисковой системы Google. По инициативе владельца на его сайте размещаются текстовые и графические объявления, соответствующие контексту поискового запроса. За показ этих рекламных блоков, а так же за клики по ним владелец ресурса получает доход.
Яндекс.Метрика
Это бесплатный сервис, созданный для анализа поведения пользователей и оценки посещаемости сайтов.

Библиотеки
JQuery
Это набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML. Библиотека jQuery помогает легко получать доступ к любому элементу DOM, обращаться к атрибутам и содержимому элементов DOM, манипулировать ими. Также библиотека jQuery предоставляет удобный API для работы с AJAX.
React
Это JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов. React может использоваться для разработки одностраничных и мобильных приложений. Его цель — предоставить высокую скорость, простоту и масштабируемость. В качестве библиотеки для разработки пользовательских интерфейсов React часто используется с другими библиотеками, такими как Redux.
